# Create New Project
`Start a new Android Studio project`をクリック。
![Welcome.png](./image/Welcome.png)

`Application name`と`Company Domain`を入力してNextをクリック。  
`Company Domain`はなんでも良いけど`Package name`になるので自分でわかりやすい名前を付ける。
